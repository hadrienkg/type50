Hello, welcome to type50!
This is just a simple typing test website, where you can procrastinate-I mean, test out your typing skills in a fun manner!

To run this project, make sure you have all of the files downloaded. Then, open this project in an IDE, code editor, etc., like VSCode. Now, in the terminal, you can run the project using the command "npm run dev". From here, you can see the project by going to this URL: "http://localhost:3000".
